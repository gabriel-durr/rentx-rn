> <h3> Getting start : </h3>
> <br/>

<div>

1. <b>Clone the project</b>

```bash
 git clone https://github.com/gabriel-durr/rentx-rn.git && cd rentx-rn
```

 <br/>

2. <b>Install dependencies</b>

```bash
 yarn
```

 <br/>

3. <b>Start the application</b>

```bash
 expo start
```
